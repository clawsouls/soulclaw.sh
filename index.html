What is SoulClaw?

SoulClaw is an OpenClaw fork built for context engineering. We took OpenClaw's powerful agent framework and added a soul layer — community-shared persona packages that give your AI agent consistent personality, memory, and behavior.

Think of it as OpenClaw + soul.md community + adaptive memory + specialized tools. Everything that makes OpenClaw great, plus the identity layer that makes your agent yours.

Built on OpenClaw (MIT). SoulClaw adds the soul ecosystem on top.

🧠 Soul Memory

4-Tier Adaptive Memory Architecture — memories that persist, decay, and protect identity
T0 — Identity Core
SOUL.md + IDENTITY.md
Immutable. Defines who your agent is. Survives across sessions, models, and platforms.
T1 — Long-Term Memory
MEMORY.md
Curated knowledge. Important decisions, people, preferences. Promoted from daily logs.
T2 — Short-Term Memory
memory/*.md
Daily session logs. Auto-compacted over time. Semantic search via bge-m3 embeddings.
T3 — Session Context
Active conversation
Current session. Automatically summarized and flushed to T2 on session end.
Less important memories decay naturally. Identity-critical ones are protected. Read more →

What SoulClaw Adds

🎭
Soul Community
Browse, install, and publish soul.md persona packages. 30+ community souls and growing.
🧠
4-Tier Adaptive Memory
Identity-aware memory that persists, decays, and promotes. Semantic search via Ollama bge-m3. Your agent remembers what matters.
🔍
SoulScan Security
55+ security rules. Detects prompt injection, PII leaks, and persona drift in 8 languages.
🔄
Swarm Memory
Git-backed memory sync across machines. Branch, merge, encrypt with age. LLM-assisted conflict resolution.
💬
Multi-Channel
Telegram, Android app, VSCode — chat with your agent from anywhere. Same soul, same memory.
🚀
Tiered Bootstrap
Load soul files progressively — saves up to 60% context tokens vs loading everything at once.
🔐
Memory Encryption
age encryption for MEMORY.md and memory files. Your agent's memories stay private, even on shared repos.
⚡
Promotion Detector
Automatically identifies important memories in daily logs and promotes them to long-term storage. No manual curation needed.
📄
Research: The Forgetting Problem
Why perfect memory breaks AI agent identity — and how declarative anchors fix it. The theory behind Soul Memory's architecture.
